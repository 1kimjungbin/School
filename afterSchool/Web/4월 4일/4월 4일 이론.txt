<img> 태그

    - 웹 문서에 이미지를 삽입할 때 사용함.
    - src 속성을 사용해 이미지 파일이 있는 경로를 알려 주어야 화면에 표시할 수 있음.

src 속성 - 이미지 파일 경로 지정

    - 이미지를 웹 문서에 넣으려면 src 속성에 정확한 파일 경로를 지정해야함.
    - 경로란 현재 HTML 문서에서 이미지 파일까지 찾아가기 위한 길

alt 속성 - 이미지를 설명해주는 대체 텍스트

    - alt 속성은 이미지를 설명하는 대체 텍스트를 삽입할 때 사용함.
    - 웹 문서에 삽입한 이미지는 시각적인 요소이기 때문에 시각 장애인들은 그 내용을 알 수 없음.
    - alt 속성을 이용하면 이미지에 대한 설명을 넣을 수 있고 화면 낭독기가 읽어줄 수 있음.
    - 대체 텍스트를 사용하면 연결 속도가 느리거나 이미지를 제대로 표현할 수 없을 때 alt 속성에 쓴 내용이 표시되어
      어떤 이미지가 사용되었는지 유추할 수 있음.
    - 블릿 이미지나 작은 아이콘처럼 특별한 의미 없이 화면을 꾸미기 위해 사용한 이미지에는 대체 텍스트를 지정하지 않아도 됨.

    ex)
    <img src="#" alt="logo">

width, height 속성 - 이미지 크기 조정하기
    - width, height 를 사용하지 않으면 원본 이미지 크기가 그대로 표시됨.
    - 작은 이미지를 크게 표시할 경우 화질이 깨질 수 있음.

    ex)
    <img src="#" alt="sky" width="250" height="250">
